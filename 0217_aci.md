## Change image

```sh
az acr list

az acr login --name containerInstancesLab302fedb58

docker build -t containerinstanceslab302fedb58.azurecr.io/container_instances_lab_3:latest . # See Dockerfile content
docker push containerinstanceslab302fedb58.azurecr.io/container_instances_lab_3:latest

az acr repository list -n containerInstancesLab302fedb58 # To check that image has succesfully been pushed

nc -lvnp 2323

az container restart -n container-instances-lab-3 -g container-instances-labs
```
